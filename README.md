# Pinball League Stats

A mobile-friendly web app for looking up pinball scores across league seasons.

---

## Folder Structure

```
pinball/
  index.html          ← The web app (don't move this)
  parse_league.py     ← Run this to convert Excel → JSON
  data/
    manifest.json     ← Lists all league JSON files (edit when adding leagues)
    spring2026.json   ← Generated by parse_league.py
    fall2026.json     ← (future seasons appear here)
```

---

## One-Time Setup

### 1. Install Python dependencies
Open a terminal in this folder and run:
```
pip install openpyxl
```

### 2. Create a GitHub account (if you don't have one)
Go to https://github.com and sign up. It's free.

### 3. Create a new GitHub repository
- Click the **+** menu → **New repository**
- Name it something like `pinball-stats`
- Set it to **Public** (required for free GitHub Pages hosting)
- Do NOT add a README (you'll push your own files)
- Click **Create repository**

### 4. Push your files to GitHub
In a terminal, navigate to this folder then run:
```
git init
git add .
git commit -m "Initial setup"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/pinball-stats.git
git push -u origin main
```
(Replace YOUR_USERNAME with your actual GitHub username)

### 5. Enable GitHub Pages
- In your GitHub repo, go to **Settings** → **Pages**
- Under "Source", select **Deploy from a branch**
- Branch: **main**, Folder: **/ (root)**
- Click **Save**
- Wait ~2 minutes, then your site is live at:
  `https://YOUR_USERNAME.github.io/pinball-stats/`

### 6. Add to Home Screen on your phone
- Open the URL above in Safari (iPhone) or Chrome (Android)
- iPhone: tap the Share icon → **Add to Home Screen**
- Android: tap the menu → **Add to Home Screen**
- It will appear as an app icon on your home screen

## Adding a New League Season

### Step 1 — Edit parse_league.py
Open `parse_league.py` and change the top two CONFIG lines:
```python
LEAGUE_NAME = "Fall 2026"              # ← new league name
EXCEL_FILE  = "League_RawData.xlsx"   # ← your Excel filename
```
Make sure your Excel file is in the same folder as the script.

### Step 2 — Run the parser
```
python parse_league.py
```
This creates `data/fall2026.json` automatically.

### Step 3 — Update the manifest
Open `data/manifest.json` and add the new file:
```json
{
  "files": [
    "data/spring2026.json",
    "data/fall2026.json"
  ]
}
```

### Step 4 — Push to GitHub
```
git add .
git commit -m "Add Fall 2026 league data"
git push
```
Your phone will see the new data within seconds (just refresh the app).

## Fixing a File (like the Sheet2 duplicate issue)

1. Fix your Excel file
2. Run `python parse_league.py` again (it overwrites the old JSON)
3. Push to GitHub:
   ```
   git add .
   git commit -m "Fix Spring 2026 data"
   git push
   ```

## Using the App

1. Open the app URL on your phone (or tap the home screen icon)
2. Tap ⚙ Settings and enter your name as **Last, First** (e.g. `Damel, Scott`)
   - Your scores will always appear alongside whoever you look up
   - Theresa can do the same on her phone and set her own name
3. Type any player's name in the search box
4. Tap their name from the dropdown
5. Each machine shows: **High | Average | Low**
6. If you've played the same machine, your stats appear alongside for comparison
7. Tap any machine card to expand it
8. Tap **Show Raw Scores** to see week-by-week scores

## Notes

- The app works offline after first load (cached in your browser)
- Data is public (anyone with the URL can see it) — fine for league scores
- No app store required; it installs directly from the browser
- Scores across multiple leagues accumulate automatically once added to manifest.json
